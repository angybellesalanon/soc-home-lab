Objective:
This project consists of building a SOC (Security Operations Center) home lab using VMware to simulate real-world cyber attacks and practice detection and incident response.
Environment Setup
Virtualization: VMware
Attacker Machine: Kali Linux
Target Machines: Windows 10, Ubuntu
SIEM: Wazuh
Activities Performed
Network scanning using Nmap
Identification of exposed services
Simulation of attacks:
Brute force attack
Log collection and correlation using Wazuh SIEM
Detection of malicious activity
Generation of security alerts
Log analysis and incident investigation
Basic incident response actions

SOC Capabilities Demonstrated
Intrusion detection
Log analysis
Threat identification
Alert monitoring
Incident response
